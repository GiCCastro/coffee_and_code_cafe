# Coffee & Code Café

Uma landing page responsiva criada como projeto de portfólio em frontend e conteúdo.

Este projeto representa um café fictício pensado para desenvolvedores, estudantes e criativo, combinando um ambiente acolhedor com produtividade e criatividade.  
Também demonstra documentação bilíngue (EN / PT-BR).

---

## Visão Geral do Projeto

Coffee & Code Café é uma landing page responsiva desenvolvida para demonstrar:

- Habilidades em desenvolvimento frontend
- Layouts web limpos e bem estruturados
- Design responsivo para diferentes dispositivos
- Conteúdo web claro e natural
- Integração entre design, conteúdo e experiência do usuário

Este projeto foi desenvolvido como peça de portfólio e não representa um negócio real.

---

## Tecnologias Utilizadas

- HTML5  
- CSS3 (Flexbox & Grid)
- Técnicas de layout responsivo
- Estrutura semântica e noções básicas de acessibilidade

---

## Conteúdo & Tradução

O conteúdo do site foi escrito em inglês.

Ferramentas de IA foram utilizadas como parte do fluxo de trabalho para auxiliar na formulação e tradução dos textos.  
Toda a revisão final, adaptação de conteúdo, estrutura e decisões de implementação foram realizadas por mim.

Este projeto também serve como exemplo de:
- Estruturação de conteúdo para web
- Organização de texto com foco em UX
- Consciência de tradução e localização

---

## Responsividade

A landing page é totalmente responsiva e se adapta a:

- Telas de desktop
- Tablets
- Dispositivos móveis

Layouts, espaçamentos e imagens foram ajustados para garantir boa legibilidade e usabilidade em diferentes tamanhos de tela.

---

## Objetivo

Este projeto foi criado para fins de aprendizado e portfólio, destacando habilidades em:

- Desenvolvimento frontend
- Design responsivo
- Conteúdo orientado para web
- Consciência de tradução
