# Coffee & Code Café

A responsive landing page created as a frontend and content-focused sample project.

This project represents a fictional café designed for developers, students, and creatives. Combining a cozy atmosphere with productivity and creativity.  
It also demonstrates bilingual documentation (EN / PT-BR).

---

## Project Overview

Coffee & Code Café is a responsive landing page built to demonstrate:

- Frontend development skills  
- Clean and structured web layouts  
- Responsive design across devices  
- Clear and natural web content in English  
- Integration of design, content, and user experience  

The project was developed as a portfolio piece and does not represent a real business.

---

## Technologies Used

- HTML5  
- CSS3 (Flexbox & Grid)  
- Responsive layout techniques  
- Basic accessibility and semantic structure  

---

## Content & Translation Notes

The website content is written in English.

AI tools were used as part of the workflow to support wording and translation.  
All content decisions, revisions, and final structure were defined by me.

This project also serves as a sample for:

- Web content structure  
- UX-focused text organization  

---

## Responsiveness

The landing page is fully responsive and adapts to:

- Desktop screens  
- Tablets  
- Mobile devices  

Layouts, spacing, and images were adjusted to ensure readability and usability across different screen sizes.

---

## Purpose

This project was created for portfolio and learning purposes, showcasing skills in:

- Frontend development  
- Responsive design  
- Web-oriented content  
- Translation awareness
